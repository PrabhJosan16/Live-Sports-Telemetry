 Live Telemetry App

Use Dummy data --> .env in the backend and set USE_DUMMY=true, to run --> node src/index.js


Use Dummy data --> .env in the backend and set USE_DUMMY=false,SESSION_PREFERENCE=latest 
(latest - Most recent session (recommended for testing)
current - Only sessions happening right now
race - Most recent race session specifically)

to run backend: npm run dev
to run frontend: npm run dev
